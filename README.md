# Orienta
Orientador escolar que busca soluciones a diferentes casos de alumnos y familias con dificultades. Herramientas, pautas, soluciones, idea...

## Estructura
- [`/casos`](casos/): casos típicos con pautas de actuación
- [`/recursos`](recursos/): enlaces, contactos y materiales de apoyo
- [`/plantillas`](plantillas/): plantillas reutilizables (informes, casos nuevos...)
- [`/herramientas`](herramientas/): pequeñas utilidades HTML/JS que se abren
  en el navegador (sin instalación ni servidor)
- [`CLAUDE.md`](CLAUDE.md): contexto del proyecto para futuras sesiones
