# Plantillas

- [`plantilla-caso.md`](plantilla-caso.md): para describir un **tipo de
  caso general** (ej. acoso escolar) con pautas de actuación, pensado para
  guardarse en [`/casos`](../casos/).
- [`plantilla-seguimiento-individual.md`](plantilla-seguimiento-individual.md):
  para el **seguimiento de un alumno/a concreto** (registros periódicos,
  reuniones de coordinación, traspaso entre cursos). Rellenarla siempre de
  forma anonimizada si el archivo resultante se va a guardar en este
  repositorio; si contiene datos identificativos, mantenerla fuera del repo.
- [`plantilla-informe-fin-curso.md`](plantilla-informe-fin-curso.md): para
  el **informe de cierre de curso** de un alumno/a con seguimiento
  individualizado (tabla histórica de cumplimiento, valoración por
  ámbitos, comunicación a la familia y plan de traspaso al curso
  siguiente). Misma norma de anonimización/privacidad que la plantilla
  anterior.
