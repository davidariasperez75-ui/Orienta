Seguimiento semanal

Registra la semana y obtén al instante un resumen visual y un mensaje listo para enviar a la familia

Los datos se guardan solo en este navegador/dispositivo (no se suben a ningún sitio). Usa un alias para cada caso (no es necesario el nombre real) y exporta una copia de vez en cuando para no perder el historial.

Configuración del caso (ámbitos, indicadores y objetivo)

El mensaje a la familia solo repite una propuesta para un ámbito si cambia de nivel (Alta/Media/Baja) o si el cumplimiento varía al menos estos puntos respecto a la semana anterior. Si un ámbito lleva varias semanas estancado por debajo del objetivo, se avisa cada 3 semanas.

Cada ámbito tiene una lista de indicadores observables (uno por línea). Cada semana se marca si se cumplió o no cada indicador.
